# @rune-langium/visual-editor

> Render and edit Rune DSL models with ReactFlow-based graph, form, and store components.

Browser UI components and helpers for graph-based Rune DSL editing.

## Features

- Render interactive graphs with `RuneTypeGraph`
- Embed namespace, detail, and editor panels in custom shells
- Reuse layout, validation, and store helpers in Studio-style apps
