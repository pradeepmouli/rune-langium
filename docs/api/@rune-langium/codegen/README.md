# @rune-langium/codegen

Code generation service and types for Rune DSL models. Provides integration with rosetta-code-generators for generating target language output from parsed Rune DSL models.

## Installation

```bash
npm install @rune-langium/codegen
```

## Usage

```typescript
import { CodegenService } from '@rune-langium/codegen';
```
