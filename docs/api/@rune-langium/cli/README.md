# @rune-langium/cli

Command-line interface for the Rune DSL, powered by [Langium](https://langium.org).

## Installation

```bash
npm install -g @rune-langium/cli
```

## Usage

```bash
# Parse and validate Rune DSL files
rune-dsl parse <file.rosetta>

# See all available commands
rune-dsl --help
```
