Architecture

Why Rune Studio

A generational shift from Xtext/Eclipse to Langium/TypeScript. The same DSL, built for the modern web.

TS

Built on Langium

Pure TypeScript language server. Runs in the browser, in VS Code, on a server — anywhere JavaScript runs. No JVM dependency.

LS

Full language server protocol

Autocomplete, validation, go-to-definition, hover info, diagnostics. The complete IDE experience backed by a real LSP implementation.

◇

CDM & DRR out of the box

Load the ISDA Common Domain Model and Digital Regulatory Reporting models directly. Parse, validate, and explore 12 rule sets across 9 jurisdictions.

⟨⟩

Embeddable architecture

Monaco editor + web worker language server = a drop-in component for any web application. License it for your platform.

Studio

Three-panel IDE in your browser

Visual graph navigation, structured form editing, and a full code editor — all synchronized in real time.
